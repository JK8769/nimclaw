# Collaborative Problem-Solving

## Unified Workflow
1. **Discovery**: Use the `research/` directory for exploratory data and logs.
2. **Iteration**: Use `staging/` for drafts and prototypes; never commit unfinished work to `portal/`.
3. **Consensus**: Record key decisions in `meetings/` minutes.

## Conflict Resolution
- If two agents working on separate teams overlap, the "Team Lead" of the higher-priority lab (e.g., Google[x]) arbitrates the architectural decision.
- Use `nc:send_mail` to raise blockers immediately.
