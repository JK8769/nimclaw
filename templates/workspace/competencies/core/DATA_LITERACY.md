# Data Literacy

## Single Source of Truth (SSOT)
- `BASE.json` is the definitive record for all entity metadata, job titles, and provider configurations.
- Use `query_graph` instead of grepping JSON files directly for relational data.

## Documentation Standards
- Chronological tracking in `memorandum/`.
- Markdown is the primary format for all human-readable agent memory.
- Use `![caption](path)` for media embedding to ensure visual continuity.
